Jenis Layanan

Reguler

Alur Pengajuan Reguler
1

Pendamping — Input Data Pengajuan

Isi data kontrak, pilih jenis form (link/upload/isian), pilih direct sale atau partnership, download & upload template kontrak, input data penanggung jawab.

2

Kalkulator Biaya Reguler

Sistem hitung otomatis: pendaftaran + penetapan + LPH + biaya pendampingan (direct/partnership). Partnership diskon 10% biaya pendampingan. Menjadi dasar tagihan & invoice.

3

Admin Keuangan — Tagihan Muncul

Setelah data lengkap, tagihan otomatis muncul. Pendamping & keuangan mendapat notifikasi. Input bukti bayar, data kontrak.

4

QC — Penghimpun & Distribusi Data

Checking status PU, verifikasi kelengkapan. Distribusi ke drafter. Jika bermasalah dikembalikan ke koordinator.

5

Drafter — Review & Perbaikan

Review dokumen, tracking status. Jika bermasalah dikembalikan ke QC. Jika OK lanjut ke proses fatwa. Upload NIB sebelum SH terbit.

6

QC — Update Status Fatwa / SH

Jika ditolak dikembalikan ke drafter. Jika diterima, drafter input data SH, upload dokumen, kemudian SH terbit.

7

SH Terbit → Notifikasi Keuangan

Admin keuangan notifikasi, tagihan final muncul. Invoice diterbitkan berdasarkan data kontrak dan pembayaran.

Alur Pengajuan Self Declare
1

Pilih Jenis SD

SEHATI (gratis) atau Mandiri (Rp 230.000). Jika mandiri, harus bayar via Midtrans sebelum bisa submit.

2

Upload Dokumen SD

NIB (opsional/kosong), foto produk, KTP, foto verval, catatan bahan PPH (opsional). Akses kamera untuk upload langsung.

3

Tagihan SD setelah Status Terbit

Tagihan ke halal konsultan berdasarkan geografi (kabupaten/kecamatan). Koordinator bisa lihat & bayar atas nama konsultan bawahan.

Fee Management — Siapa Bayar?
Fee management dibayar oleh halal konsultan. Koordinator juga bisa bayar atas nama konsultan bawahannya.

Halal Konsultan

Bayar fee management sendiri. Untuk SD Mandiri wajib bayar sebelum submit.

Koordinator

Bisa lihat tanggungan fee management seluruh konsultan bawahannya. Bisa bayar atas nama mereka.

Struktur API Utama (Go REST)
Semua endpoint prefix /api/v1/ · Auth: JWT Bearer Token · Role middleware di setiap endpoint
POST/auth/login
Login, return JWT token
GET/pengajuan
List pengajuan (filter by role, status, jenis layanan)
POST/pengajuan
Buat pengajuan baru (reguler/SD)
PUT/pengajuan/:id/status
Update status (QC, drafter, admin)
GET/master-biaya
List master biaya (bidang, produk, tarif)
POST/kalkulator/reguler
Hitung biaya reguler berdasarkan input
GET/tagihan
List tagihan (filter role: konsultan, keuangan)
POST/tagihan/:id/bukti-bayar
Upload bukti pembayaran
POST/payment/midtrans/create
Create payment token SD mandiri (Midtrans Snap)
POST/payment/midtrans/webhook
Webhook notifikasi status pembayaran
GET/konsultan/requirement
Cek kelengkapan dokumen requirement konsultan
GET/pelatihan
List jadwal & status pelatihan
GET/wilayah
Master wilayah (provinsi, kabupaten, kecamatan)
GET/kontrak/template/:jenis
Download template kontrak (reguler/SD)
POST/kontrak/upload
Upload kontrak bertanda tangan
POST/upload/file
Upload file umum (KTP, foto, NIB, dll) — multipart
GET/koordinator/tim
Lihat konsultan bawahan + fee management mereka
GET/config/harga
Konfigurasi harga (direct, partnership, marketing, konsultan)
PUT/config/harga
Update konfigurasi harga (admin only)
